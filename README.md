# Comfortbilt HP22 Remote / LIRC Config
LIRC config and scripts to simulate the IR remote of a Comfortbilt HP22 stove

## Key Mappings

![Remote](remote.png)


| Remote | LIRC             |
|--------|------------------|
| ON     | KEY_TOUCHPAD_ON  |
| OFF    | KEY_TOUCHPAD_OFF |
| MANUAL | BTN_0            |
| TEMP   | BTN_1            |
| UP     | KEY_UP           |
| DOWN   | KEY_DOWN         |
| HOLD   | BTN_2            |
| UNHOLD | BTN_3            |
